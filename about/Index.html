Генератор тегов и жанров

Задача: Динамически обновлять теги и жанры для медиафайлов на основе доступной базы.

Решение: Система случайного, но уникального подбора тегов из JSON-базы данных.

// Генерация случайных тегов и жанров
const randomTags = getRandomItems(uniqueData.tags, 4, 9);
const randomGenres = getRandomItems(uniqueData.genres, 6, 9);
